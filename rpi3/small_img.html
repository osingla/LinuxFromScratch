This guide explains step by step how to create a small linux image for the Rasperry PI 3, using buildroot and a Linux host system (such as Fedora, Debian or Ubuntu).

  • The kernel is armv8 (64-bits).
  • We use glibc, and not uCLibc.
  • We will use a small initramfs image.
  • All files are stored in a single fat32 partition of the uSD card.
