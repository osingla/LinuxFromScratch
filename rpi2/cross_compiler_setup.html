This show how to setup the cross-compiler, as an example:
~$
~$ cd ~/lfs/rpi2/min_img/buildroot-2015.11.1
~/lfs/rpi2/min_img/buildroot-2015.11.1$ svn export https://github.com/osingla/LinuxFromScratch/trunk/hello_world
~/lfs/rpi2/min_img/buildroot-2015.11.1$ cd hello_world/
~/lfs/rpi2/min_img/buildroot-2015.11.1/hello_world$ make
You will have a hello program
~/lfs/rpi2/min_img/buildroot-2015.11.1/hello_world$ file hello
hello: ELF 32-bit LSB executable, ARM, EABI5 version 1 (SYSV), dynamically linked, interpreter /lib/ld-uClibc.so.0, not stripped
~/lfs/rpi2/min_img/buildroot-2015.11.1/hello_world$
Just scp this hello program into the target to test it.
