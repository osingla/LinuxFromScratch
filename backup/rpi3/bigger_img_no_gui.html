This step describes how to build an image for the Rasperry PI2 with more packages, but still no GUI.

Create a working directory on the Linux host PC, for instance:
~$ cd lfs
~/lfs$ cd rpi2/
~/lfs/rpi2$ mkdir bigger_img_no_gui/
~/lfs/rpi2$ cd bigger_img_no_gui
~/lfs/rpi2/bigger_img_no_gui$ wget --no-check-certificate https://buildroot.uclibc.org/downloads/buildroot-2015.11.1.tar.bz2
~/lfs/rpi2/bigger_img_no_gui$ tar -f buildroot-2015.11.1.tar.bz2 -x
~/lfs/rpi2/bigger_img_no_gui$ cd buildroot-2015.11.1/
~/lfs/rpi2/bigger_img_no_gui/buildroot-2015.11.1$ wget --no-check-certificate \
https://raw.githubusercontent.com/osingla/LinuxFromScratch/master/configs/rpi2/bigger_img_no_gui/.config
~/lfs/rpi2/bigger_img_no_gui/buildroot-2015.11.1$ make
Create then a uSD card:
~/lfs/rpi2/bigger_img_no_gui/buildroot-2015.11.1$ wget https://raw.githubusercontent.com/osingla/LinuxFromScratch/master/create-sdcard.sh ~/lfs/rpi2/bigger_img_no_gui/buildroot-2015.11.1$ ./create-sdcard.sh --device /dev/sdg
